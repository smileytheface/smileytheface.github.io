Address Dispenser
Angular | Node.js
Express.js | MongoDB
Address Dispenser is a full stack web application for dispensing addresses to letter writers during the pandemic with full CRUD functionality. It uses a third party api to text/email addresses to writers, and keeps track of which writers have which addresses assigned to them. It is currently in progress, but you can watch a sample of it on youtube, and/or view the angular code on github.
